Cantinarr is the companion app for a Cantinarr server — a self-hosted media request and library manager for households that run their own media setup with Radarr, Sonarr, or Chaptarr behind it.

You need a Cantinarr server to use this app. The server is open source and runs as a single Docker container; setup instructions are on GitHub.

FOR YOUR HOUSEHOLD

Everyone you share the server with gets a clean way to find and request things:

• Browse trending and popular movies and shows, powered by TMDB and Trakt
• Search everything from one bar, with plain availability labels: Available, Requested, Downloading
• Request in one tap, or pick exactly which seasons (or book formats) you want
• Watch download progress live, and get a push notification when your request is approved and when it's ready
• Something wrong with a file? Report a problem right on the title and the server's AI agent can investigate
• Ask the built-in AI assistant what to watch tonight; if your admin selects ChatGPT (Codex), link your own ChatGPT account with a one-time browser code

No accounts to create and no passwords to remember: your admin sends a connect link, you open it, you're in. Passkey and password sign-in are there when an admin enables them.

FOR ADMINS

The management side goes as deep as you want:

• Approve or deny requests from the queue or straight from the push notification, with per-user policies for seasons and quality
• Drill into Radarr and Sonarr: series → season → episode, batch searches, Edit Series, interactive release search
• One download queue across SABnzbd, qBittorrent, NZBGet, and Transmission
• Import Doctor: stuck downloads explain themselves in plain English, with one-click fixes
• See what's playing through Tautulli — active streams, history, top stats
• Plex invites, a guided setup checklist, per-user defaults, and device management
• Push notifications with a single line of server config — no Apple developer account, certificates, or Firebase project on your side
• Your server is an MCP server too: connect Claude or any MCP client and manage your library with the same tools that power the built-in assistant

Works on iPhone and iPad, with a sidebar layout on bigger screens.

PRIVACY

The app connects to your own server. API keys stay server-side, availability is computed live from your services, and there are no ads, analytics, or tracking. Poster art loads from the TMDB image CDN. When you use the assistant, your server sends its prompt and context to the AI provider your admin selected. If you choose to link ChatGPT, the app opens OpenAI's sign-in page in your browser; your password never passes through Cantinarr.
